
# Laravel 5.4 - Banco de Dados e Eloquent

https://www.udemy.com/course/laravel-54-banco-de-dados-e-eloquent/

Aprenda a Trabalhar com Banco de Dados no Laravel 5.4. Vamos abordar o trabalho com múltiplos banco de dados, Query Builder e o Eloquent com relacionamentos entre as tabelas. E ainda, você vai aprender a configurar o sistema com o Laravel 5.4 para reconhecer um Banco de Dados já modelado configurando os seus relacionamentos entre as tabelas.
